Welcome to the Dragon Operating System Page

What is the Dragon Operating System

Right Now, DragonOS is little more than an idea and a simple interface that's not nearly complete. Eventually however, I would love to see it become a possible option for building simple AVR applications across many differing devices and peripherals.

What is the Meggy Jnr RGB

It's this thing that you can buy and then build and hopefully play and if your really clever (or required to by your lecturer) program it. You can find heaps more about it at EvilMadScientists.

Quick Notes

Building the OS

At the moment it isn't working, but eventually (I'll update you on when) the following commands will be used to first build and then push the code onto the Meggy. 

make
make push

And now the OS is on the Meggy you can get to work and either play the built in game or start to build your own.
